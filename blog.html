Some Common Questions

Q-1: What is DOM? What is the purpose of DOM?

Answer: DOM is a Document Object Model. It's basically, used to allows programs to dynamically access and update the content, structure, and style of a document in html.

Q-2: How will you select HTML elements using DOM? Name the DOM methods.

Answer: There are some ways in which I can select HTML elements in a DOM using selectors:-
1. getElementsByTagName(); 2. getElementsByClassName(); 3. getElementById(); 4. querySelector(); 5. querySelectorAll();

Q-3: What is event bubble?

Answer: Event bubble is a kind of event propagation in DOM, where the event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

Q-4: What is a callback function and why will you use it?

Answer: A callback function is a function that passed as an argument to another function and can return back without any parameters. I use it to get elements of HTML by calling and create or remove any type of elements what normally do not exist in the HTML.
